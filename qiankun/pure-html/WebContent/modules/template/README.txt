文件命名规则：
    模块名+功能+类别.html
    如：templateReadOnlyForm.html
    模块名：根据业务确定，尽量唯一
    功能名：
        只读页：ReadOnly
        添加和修改页：Edit
        统计页：Count
      说明：这个根据具体功能定义，但尽量做到统一

    类别：
        表单：Form
        列表：List

    功能或类别没有可以省略，如统计页：templateCount.html
